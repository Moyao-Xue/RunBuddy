# RunBuddy - 你的跑步伙伴

> 面向新手跑者的游戏化跑步伴侣 🐶

## 项目简介

RunBuddy 是一款专为跑步初学者设计的 Web App，通过游戏化的方式让跑步变得更有趣。玩家可以与可爱的虚拟伙伴一起跑步，赚取金币，解锁装扮，探索苏州经典跑步路线。

## 功能特色

### 首页
- 可爱的卡通狗虚拟伙伴 🐶
- 实时显示今日跑步距离、累计金币、总里程
- 模拟跑步功能，点击开始后伙伴会有跳跃动画
- 跑步时实时更新距离和预计金币收益
- 每公里可获得10金币奖励

### 地图页
- 预设苏州经典打卡点：
  - 🏯 寒山寺（累计5公里解锁）
  - 🌉 枫桥（累计10公里解锁）
  - 🏮 山塘街（累计15公里解锁）
- 路线解锁进度条
- 地图功能开发中占位

### 装扮页
- 商店系统，可购买各种装扮：
  - 🧢 帽子 - 100金币
  - 🕶️ 墨镜 - 150金币
  - 👕 T恤 - 200金币
  - 👟 跑鞋 - 250金币
- 已购买装扮显示"已拥有"
- 金币不足时按钮禁用

### 我的页
- 用户头像展示（带装备显示）
- 累计跑步总里程
- 累计获得总金币
- 已解锁装扮数量
- 已解锁路线数量
- 成就系统：
  - 🎯 首次开跑
  - 🏅 5公里达成
  - 💰 金币达人

## 技术栈

- **HTML5** - 语义化页面结构
- **CSS3** - 现代化渐变主题、响应式设计
- **Vanilla JavaScript** - 原生 JS，无框架依赖
- **localStorage** - 数据持久化存储
- **Google Fonts** - Nunito 字体

## 快速开始

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 开始你的跑步之旅！

## 数据说明

### 初始数据
- 初始金币：500
- 初始今日距离：0
- 初始总里程：0
- 初始装扮：空

### 金币规则
- 跑步收益：1公里 = 10金币
- 购买装扮会消耗金币

### 路线解锁
| 路线 | 解锁条件 |
|------|----------|
| 寒山寺 | 累计跑步5公里 |
| 枫桥 | 累计跑步10公里 |
| 山塘街 | 累计跑步15公里 |

## 响应式设计

- 最大宽度 450px 居中显示
- 按钮最小高度 44px（符合移动端触控标准）
- 自适应各种屏幕尺寸
- 支持安全区域适配（iPhone刘海屏）
