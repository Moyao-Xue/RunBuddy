## 生成说明

1. **目录结构创建**

2. **index.html** - 主页面结构

   - 包含 4 个页面 sections（首页、地图、装扮、我的）
   - 底部固定导航栏

   - 引入所有 JS 模块

3. **css/style.css** - 样式文件
   - 定义 CSS 变量管理主题色

## 技术实现要点

### 模块化架构
```
Store (数据持久化)
    ↓
User (用户数据) ← → Pet (虚拟伙伴)
    ↓               ↓
Map (地图)       App (主应用)
```

### 数据流
1. 用户操作 → App 接收
2. App 调用对应模块处理
3. 模块更新数据到 Store (localStorage)
4. UI 响应式更新

### 跑步模拟
- 使用 `setInterval` 模拟跑步
- 随机速度生成 (6-10 km/h)
- 实时更新距离和金币

### 成就系统
- 首次开跑：完成至少 1 公里
- 5 公里达成：今日距离达到 5 公里
- 金币达人：累计金币达到 1000
